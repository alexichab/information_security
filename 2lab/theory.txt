1. Шифр Шамира

Шифр Шамира реализует схему разделения секрета с использованием модульной арифметики. Два участника (А и Б) выполняют два последовательных этапа шифрования и расшифрования.
Основные этапы:

Шаг 1 (Шифрование):Два участника (А и Б) выбирают свои секретные ключи k1 и k2.

    Сообщение m сначала шифруется с использованием ключа k1 первого участника, результат обозначим как c1 = m^k1 mod p.
    Второй участник применяет свой ключ k2, чтобы получить зашифрованное сообщение: c2 = c1^k2 mod p.

Шаг 2 (Расшифрование):Каждый участник применяет свою инверсию ключа к зашифрованному сообщению:Второй участник применяет инверсию своего ключа: d1 = c2^(k2^-1) mod p.

    Первый участник применяет инверсию своего ключа: d2 = d1^(k1^-1) mod p, восстанавливая исходное сообщение.


    k1, k2 — случайные ключи участников.
    Сообщение шифруется сначала с использованием k1, потом с использованием k2.
    Для расшифрования сначала используется инверсия ключа второго участника, потом инверсия ключа первого участника.



2. Шифр Эль-Гамаля

Шифр Эль-Гамаля основан на сложности вычисления дискретного логарифма. Секретный ключ хранится у владельца, а открытый ключ используется для шифрования.
Основные этапы:

Шаг 1 (Шифрование):Выбирается случайное число k для каждого зашифрованного сообщения.

    Вычисляются два компонента: a = g^k mod p и b = (m * y^k) mod p, где y = g^x mod p — открытый ключ.

Шаг 2 (Расшифрование):Используется секретный ключ x, чтобы вычислить сообщение как m = b / a^x mod p.


    x — секретный ключ, y = g^x mod p — открытый ключ.
    Для шифрования используются случайные значения и два компонента.
    Расшифрование происходит с использованием секрета x, что позволяет восстановить сообщение.


3. Шифр Вернама

Шифр Вернама — это симметричный шифр, использующий ключ той же длины, что и сообщение. Операция XOR применяется побитово между байтами сообщения и ключа.
Основные этапы:

Шаг 1 (Шифрование): Для каждого байта сообщения выполняется операция XOR с соответствующим байтом ключа.

Шаг 2 (Расшифрование): Операция XOR снова выполняется, чтобы восстановить исходное сообщение.

    Создается случайный ключ длины сообщения.
    Байты сообщения и ключа побитово XOR-ятся для шифрования.
    Для расшифрования выполняется тот же процесс с тем же ключом.



4. Шифр RSA

RSA — это асимметричный алгоритм, где каждый участник имеет два ключа: открытый и закрытый. Открытый ключ используется для шифрования, а закрытый — для расшифрования.

    Генерация ключей:

    Выбираются два больших простых числа p и q.
    Вычисляется их произведение n = p * q, которое используется как модуль для шифрования и расшифрования.
    Вычисляется значение функции Эйлера: φ(n) = (p - 1) * (q - 1).
    Выбирается публичный ключ e — обычно небольшое целое число, взаимно простое с φ(n).
    Вычисляется закрытый ключ d как модульная инверсия числа e по модулю φ(n), то есть d * e ≡ 1 (mod φ(n)).

2. Шифрование:

    Для шифрования сообщения m используется формула: c = m^e mod n, где c — зашифрованное сообщение.

3. Расшифрование:

    Для расшифрования зашифрованного сообщения c используется формула: m = c^d mod n, где d — закрытый ключ.
